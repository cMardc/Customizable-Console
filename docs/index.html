What is "Customizable Console"?

Customizable Console project helps you to make your console based applications look better. It has builtin functions and attributes to design by yourself. You can change color, draw shapes, frames, output alert sound, show message boxes and more. It's made in C and can also be used in C++. Installation is very easy, just copy the header file and you're ready to go!
